This project uses Machine Learning to scan different pills and identify them. When you first login into the app it takes you to a page where you can manually input your medications and get reminders to take them. You can also identify lose pills by taking a picture of the pill and it will tell you what it is. This increases saftey and reduces waste. We used machine Learning to scan different pills and identify them. There is a also chat box you can use if you have questions about certain medication. Every year, millions misuse OTC meds by mixing them or taking outdated pills. PillPal puts trustworthy information at your fingertips.
